### global installation of typescpit:

```
npm i typescript -g
```

### when PS dont want to execute

```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### compile

```
tsc main.ts
```
### watch changes on file and recompile

```
tsc main.ts -w
```
### init tsconfig.json

```
tsc --init
```
